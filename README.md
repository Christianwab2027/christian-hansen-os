# Insight OS

A Brian Jung OS-style personal homepage, tailored to Christian Hansen, The Insight, InsightLRN, Insight Create, Instagram, LinkedIn, YouTube, and the full about page.

## Run locally

```bash
cd ~/Documents/insight-os
npm install
npm run dev
```

Open http://localhost:3000.

## Important edits

Update the placeholder LinkedIn and YouTube links in:

- `app/page.tsx`
- `app/about/page.tsx`

Then test:

```bash
npm run build
```
